📖 关于酒馆聊天

🍺 欢迎来到酒馆

这是一个可以与 AI 角色自由聊天的应用。你可以创建自己的角色，或与官方提供的角色互动。

🚀 快速开始

  • 创建角色：点击右上角「➕ 创建」按钮
  • 开始聊天：点击任意角色卡片
  • 模型广场：切换到「🌐 广场」发现官方角色
  • 主题切换：点击 🌙/☀️ 切换深色/浅色模式
  • API 设置：点击 ⚙️ 配置你的 OpenAI 兼容 API

💡 提示

  • 角色的背景故事和性格会影响 AI 的回复风格
  • 聊天记录保存在本地，清除浏览器数据会丢失
  • 支持 OpenAI 兼容的 API（如 OpenRouter、DeepSeek 等）
