# Spec: Tavern Chat Application

**Version:** 1.0.0
**Last Updated:** 2026-03-09

## Overview

AI 酒馆聊天应用，支持创建虚拟人并进行一对一对话。

## Functional Requirements

### FR-001: Virtual Character Creation

**Given** 用户在角色创建页面
**When** 用户填写角色信息并提交
**Then** 角色被保存到 localStorage 并出现在列表页

**Attributes:**
| Field | Type | Required | Options |
|-------|------|----------|---------|
| 名字 | string | Yes | 自由输入 |
| 性别 | enum | Yes | 男/女/其他 |
| 性格 | tags | Yes | 预设标签多选 |
| 背景故事 | text | No | 手动输入 or AI生成 |
| 自身描述 | text | No | 手动输入 or AI生成 |
| 头像 | image | No | 预设库 or 上传 |

**性格标签预设：**
- 温柔、活泼、冷漠、神秘、幽默、傲娇、成熟、可爱、毒舌、善良

### FR-002: AI-Assisted Content Generation

**Given** 用户在创建/编辑角色页面
**When** 用户点击"AI生成"按钮
**Then** 系统调用 API 生成对应内容并填充到输入框

**Scenarios:**
1. 生成背景故事：基于名字、性别、性格生成 100-200 字背景
2. 生成自身描述：基于已填信息生成 50-100 字描述

### FR-003: Character List Management

**Given** 用户在角色列表页
**When** 用户执行列表操作
**Then** 系统响应相应功能

**Features:**
- 名称搜索：模糊匹配角色名
- 标签筛选：按性格标签过滤
- 编辑：跳转到编辑页，预填现有数据
- 删除：二次确认后删除

### FR-004: One-on-One Chat

**Given** 用户选择了一个角色进入聊天
**When** 用户发送消息
**Then** AI 以角色身份回复

**Behavior:**
- 流式输出（逐字显示）
- 自动滚动到最新消息
- 历史记录持久化（localStorage）
- 角色设定注入 system prompt

### FR-005: Chat History Persistence

**Given** 用户与角色有聊天记录
**When** 用户刷新页面或重新进入
**Then** 历史消息完整显示

**Storage Schema:**
```javascript
{
  characterId: string,
  messages: [
    { role: 'user', content: string, timestamp: number },
    { role: 'assistant', content: string, timestamp: number }
  ]
}
```

## Non-Functional Requirements

### NFR-001: Performance
- 首屏加载 < 2s
- 消息发送响应 < 500ms（首字）
- UI 交互无卡顿

### NFR-002: Usability
- 移动端适配（响应式）
- 清晰的视觉层级
- 一致的交互模式

### NFR-003: Reliability
- API 错误有友好提示
- 数据自动保存
- 网络断开时保留本地数据

## UI Structure

```
/               → 角色列表页（首页）
/create         → 角色创建/编辑页
/chat/:id       → 聊天页
```

## API Integration

**Endpoint:** 阿里百炼 GLM-5
**Auth:** API Key（从环境变量或配置读取）

**Request Format:**
```javascript
{
  model: 'glm-5',
  messages: [
    { role: 'system', content: '角色设定...' },
    { role: 'user', content: '用户消息' },
    ...
  ],
  stream: true
}
```

## UI/UX Design Guidelines

- **风格:** 现代酒馆氛围（暖色调、木质纹理、柔和灯光）
- **布局:** 单栏居中，最大宽度 768px
- **字体:** 系统默认，标题用衬线体增加质感
- **动画:** 轻微过渡，不过度装饰
