## State Management

```javascript
// 全局状态
const state = {
  currentPage: 'home',      // home | create | chat
  characters: [],           // 角色列表
  currentCharacter: null,   // 当前聊天角色
  chatHistory: {},          // { characterId: messages[] }
  filters: {
    search: '',
    tags: []
  }
};
```

## API Flow

```
用户发送消息
    ↓
获取当前角色设定
    ↓
构建 messages 数组
    ├── system prompt (角色设定)
    └── 历史消息 (最近 N 条)
    ↓
调用 API (stream: true)
    ↓
逐字显示回复
    ↓
保存到 localStorage
```

## System Prompt Template

```
你是{角色名}，一个{性别}虚拟角色。

【性格】
{性格标签}

【背景故事】
{背景故事}

【自我描述】
{自身描述}

请以第一人称回应，保持角色一致性。
```

## Error Handling

| Error | User Message | Action |
|-------|--------------|--------|
| API 超时 | "回复生成中，请稍候..." | 重试按钮 |
| API 错误 | "出了点问题，请重试" | 重新发送选项 |
| 存储满 | "存储空间不足，请清理历史" | 引导清理 |
| 网络断开 | "网络已断开，消息已保存" | 离线模式提示 |
